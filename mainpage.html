Explain the .reduce() method

arr.reduce(function(prev,cur,index,arr){},init)



.reduce() callback 4 things

prev: previous value, an accumulator, tracks the last number that is handled

cur: current value, the number is now in the function

index: the index of current value

arr: the arrays you use


.reduce() can 2 arguments


one is the function, the other is initial value

if you don't put initial value there, it will automatically use arr[0] as the initial value.

.reduce() method is kind like .forEach(), but does more things



var sum =arr.reduce(function(accu,curr){
return accu+curr
})


this is how I culculate the sum of the arrays in js. In the function, I return 'accu+curr' , in the next term, "accu+curr" becomes next accu, it will keep adding the next number in the arrays untill the last one
